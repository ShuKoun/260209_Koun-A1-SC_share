**Benchmark S｜SC 极限测试 v1.4.6-017（MegaUltra2，3-Cell Threshold）阶段报告**。

# 一、阶段目标与方法回顾

本阶段的核心问题是：

> 在 v1.4.6-015（Ultra，240×120）与 v1.4.6-016（SuperUltra，480×240）中，
> 0.5nm 几何是否仍被离散 aliasing 屏蔽？

为此我们执行了 **v1.4.6-017（MegaUltra2，640×320）**，
目标是跨越“3-Cell Rule”：

* Lx = 100nm
* Nx = 640
* dx ≈ 100 / 639 ≈ 0.156 nm
* SlotW = 0.5 nm
* SlotW / dx ≈ 3.2

这意味着 0.5nm 几何特征已经覆盖超过 3 个网格单元，
aliasing 解释应被排除。

所有其他参数保持严格锁定：

* C4 Only
* SlotW = 0.5 nm
* BiasMax = 12 V
* RelayBias = 12 V
* Q_trap = 3e19
* Alpha = 0.00
* Solver / Harness / Schema / Budget / Step List 均未改动

v1.4.6-017 数据统计：

  * CONV = 50

  * BUDGET_TIMEOUT = 0

结论：

> 在 3-cell 阈值条件下，系统仍然完全收敛。

# 四、关键结论

### 1️⃣ 3-Cell Rule 已被满足

0.5nm 几何在 v017 下已被完整解析。
aliasing 不再是主导解释。

### 2️⃣ 系统确实“变硬”了

* Baseline iters 上升
* A1 iters 上升
* t_lin 近似随未知量规模增长

说明：

> 解析度提升确实增加了 PDE 刚性。

### 3️⃣ 但仍未进入破坏性分离区

* 无 GMRES_FAIL
* 无 LS_FAIL
* 无 DT_COLLAPSE
* 无 MAX_ITER
* 无 TIMEOUT

因此：

> 当前 Poisson–Boltzmann 构造族在此参数范围内仍属于强单调 / 强收敛区域。

# 五、阶段性战略判断

继续提升解析度的收益正在下降：

* 240 → 480：明显硬化
* 480 → 640：继续硬化，但仍稳定

继续堆 Nx=800 或 1000：

* 成本急剧上升
* separation 不一定出现
* 叙事增益下降

因此：

> 几何轴压测阶段基本完成。

# 六、下一阶段建议（结构诊断阶段）

建议进入 v1.4.6-018：

目标：

> 解释“为什么它不炸”。

新增 probe（不改 solver 逻辑）：

记录以下指标：

* max(|phi|)
* max(exp(|phi|/Vt))
* diag_J 的 min / median / max
* diag_pois 与指数项 term 分离统计
* A1 中 M_diag 的最小值
* 可能估计条件数 proxy

目的：

* 判断是否存在强对角占优
* 判断指数项是否受上界控制
* 确认是否存在理论上的刚性上限

如果条件数本质受限：

> 那么继续堆 Trap/Bias/网格不会打穿。

那时再考虑：

* 更极端介电跳变
* 更高掺杂比
* 或改变 PDE 结构
