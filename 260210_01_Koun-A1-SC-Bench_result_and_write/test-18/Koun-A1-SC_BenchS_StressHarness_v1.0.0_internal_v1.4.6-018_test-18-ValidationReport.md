# V. 为什么高 Bias 无法进入内点？——Screening 机制的定量解释

v1.4.6-018 给出的关键数据：

* 边界电势（phi_full_max）≈ 12–13V
* 内点最大电势（由 log_max_exp_inner 推算）≈ 0.8V

这意味着：

> 高电压几乎完全被限制在边界层附近，
> 内部电势被强烈压制。

这种现象不是偶然，而是 Poisson–Boltzmann 结构的内在属性。

---

## 1️⃣ 掺杂对比导致的强屏蔽

当前参数：

* N_high = 1e21
* N_low = 1e17
* ni = 1e10

指数项为：

[
\rho_{free} = q , ni \left( e^{-p/V_t} - e^{p/V_t} \right) + q N_{dop}
]

当 |p| 增大时，指数项迅速增长，形成强烈负反馈：

* 电势稍微上升 → 指数项迅速增加 → 产生反向电荷 → 抑制电势进一步上升

这本质上是一个“非线性恢复力”机制。

因此：

> 内点电势幅值被限制在 O(Vt × 30) 的量级，远小于边界 12V。

---

## 2️⃣ 真空槽并未破坏全域屏蔽

虽然我们引入了：

* ε discontinuity
* Q_trap
* Sub-nm 几何

但：

* 真空槽是局部区域
* 其高度仅 30nm
* 不贯穿整个域

这使得：

> 屏蔽结构仍然可以通过周围区域绕行完成。

电场未被强制进入“孤立病态区”。

# VI. 为什么 A1 的 dt 可以稳定达到 dt_max？

A1 的线性算子：

[
A = \frac{1}{dt} I - J
]

稳定条件近似取决于：

[
\frac{1}{dt} - \lambda_{\max}(J) > 0
]

在 v018 中：

* (\lambda_{\max}(J) \approx -1443)
* dt = 10 → dt^{-1} = 0.1

于是：

[
0.1 - (-1443) \approx 1443
]

这是一个极其巨大的正裕度。

这意味着：

> A1 根本不在失稳边界附近。

换句话说：

A1 不是“顽强地活下来”，
而是在“极度安全区”运行。

# VIII. 从数值现象到结构定理雏形

在 v015–v018 的连续实验中，我们得到三条可被重复验证的经验事实：

1. 提升解析度 → 刚性增加（iters/t_lin 上升）。
2. 但跨越 3-cell 阈值后仍无 failure。
3. 内点电势幅值始终受限于 O(1V)，远低于边界电压。

这三条事实共同指向一个更深层的问题：

> 当前参数族是否天然处于“强单调区”？

我们可以把这个问题形式化。

---

## 1️⃣ 强单调区的经验判定条件

基于 v018 数据，可提取三个判定指标：

### (A) 内点非线性幅值受限

[
\log \max \exp(|\phi_{inner}|/V_t) \approx 31
]

意味着：

[
|\phi_{inner}| \approx 0.8V
]

远低于外加 12V。

这是屏蔽主导结构的直接证据。

---

### (B) Jacobian 对角项保持强负值

[
\min(J_{diag}) \ll 0
]

并且：

[
|J_{diag}| \gg dt^{-1}
]

这意味着：

线性算子在对角方向具有强稳定性。

---

### (C) A1 稳定裕度巨大

[
\min(dt^{-1} - J_{diag}) \gg 0
]

A1 运行在远离不稳定边界的区域。

---

# IX. 结构解释：为什么“继续堆压”边际收益递减？

在当前构造族中，存在一个稳定机制闭环：

高边界电压
→ 指数项增强
→ 局部电荷密度迅速增加
→ 反向电场形成
→ 抑制电势进入内点

这是一种 **非线性自稳定机制**。

因此：

> 当系统处于强屏蔽区时，继续增加边界电压并不会线性增加内点非线性。

这解释了为什么：

* Bias 从 12 到 12.5V
* 解析度从 240 → 640
* SlotW = 0.5nm

都无法触发指数爆炸。
